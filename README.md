# My Safe Data

<img src="src/favicon.png">

Desktop application built with Angular, Electron e SQLite.

## Install dependencies

To install project dependencies first, run:

```bash
npm install
```

## Development (console execute)

To start a local development app, run:

```bash
npm run electron
```

## Binary (distribute)

To build a distributable binary.

```bash
npm run build-exe
```

## Download

Go to <a href="https://github.com/avmesquita/my-safe-data/releases">Releases Page</a>, download portable executable version or installer and enjoy it!
